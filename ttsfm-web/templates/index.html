Getting Started

Install TTSFM and start generating speech with just a few lines of code.

Installation 
pip install ttsfm
Requires Python 3.8+
Basic Usage 
from ttsfm import TTSClient

client = TTSClient()
response = client.generate_speech(
    text="Hello, world!",
    voice="alloy"
)
response.save_to_file("hello.wav")
